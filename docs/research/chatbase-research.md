## 3. Feature Catalog

**15+ AI Models from 6 Providers:** OpenAI (GPT-5.2, GPT-5 Mini, GPT-4o, GPT-4o Mini, GPT-4.5, O3 Mini), Anthropic (Claude Opus 4.6, 4.5, Sonnet 4.5, 3.7, 3.5), Google (Gemini 3 Pro, 2.5 Pro, 2.0 Flash), xAI (Grok 4, Grok 3), Moonshot AI (Kimi K2), DeepSeek (DeepSeek-V3, R1), Cohere (Command R+, Command R). Switch per agent at any time.

**Custom Knowledge Base Training:** Train on website URLs (automatic crawling), PDF/DOCX/TXT/CSV uploads, Notion pages, plain text, Q&A pairs, Zendesk/Salesforce support tickets. RAG-based. Daily auto-retrain option.

**AI Actions (Task Automation):** Autonomous execution of real tasks — order lookups, meeting bookings (Calendly/Cal.com), billing queries (Stripe), support tickets (Zendesk), lead capture, custom webhooks, 5,000+ apps via Zapier/Make.

**Multi-Channel Deployment:** 6 native channels — website widget (embed code/iframe), WhatsApp Business, Facebook Messenger, Instagram DMs, Slack, Email. Additional channels via Zapier, Make, REST API, webhooks.

**Lead Capture & CRM Integration:** Configure agent to collect visitor info (name, email, phone). Export as CSV. Integrate with CRMs via Zapier.

**Voice AI Agents:** Phone-based AI support (newer feature). 24/7 voice availability, natural speech responses.

**Smart Escalation:** Configure natural-language conditions for human handoff. Routes to Zendesk Sunshine, email, or helpdesk tools via integrations.

**Chat Logs & Suggestions:** Review all conversations. Confidence scores per response. "Improve Answer" feature for manual corrections. Suggestions feature spots knowledge base gaps.

**Analytics:** Conversation volume, user location, confidence scores, satisfaction ratings, topic clustering, missed question detection.

**95+ Languages:** Auto language detection. Agent responds in visitor's language even if training data is in a different language. Widget UI localized in 40+ languages including RTL (Arabic, Hebrew).

**Security:** SOC 2 Type II certified, GDPR compliant, AES-256 encryption, SSO (added Jan 2026), domain allowlisting, rate limiting, RBAC. Data never used for model training.

## 5. Key Differentiators vs FastBots.ai

Chatbase will be our second AI chatbot review under Pillar 5, making the comparison to FastBots.ai critical:

| Dimension | Chatbase | FastBots.ai |
|---|---|---|
| Starting price | $32/mo (annual Hobby) | Free tier available |
| AI models | 15+ from 6 providers | More limited selection |
| Deployment channels | 6 native + API | Website widget + API |
| AI Actions | Yes (autonomous task execution) | Limited |
| Live chat | No (integrations only) | No |
| Flow builder | No (LLM-driven) | No |
| Security certs | SOC 2 Type II + GDPR | More limited |
| Lead capture | Yes (built-in) | Limited |
| Voice support | Yes (newer feature) | No |

Positioning: Chatbase is the more advanced, higher-priced option for businesses that need intelligent AI agents with real action-taking capabilities. FastBots.ai is the entry-level, budget-friendly option for simple FAQ automation.

## 6. Key Pros & Cons (from cross-referenced sources)

### Pros
- Fastest setup in market — fully functional agent in under 10 minutes
- 15+ AI models from 6 providers — unmatched flexibility
- AI Actions system enables real task automation beyond Q&A
- SOC 2 Type II + GDPR + SSO at mid-tier pricing (rare)
- Active development — weekly feature releases in 2026
- Strong multi-channel deployment (6 native channels)
- Free plan available for testing

### Cons
- No built-in live chat for human handoff — requires external integrations
- Credit consumption varies by model (1-5 credits/response) — costs hard to predict
- White-label branding removal expensive ($99/mo + $199/mo custom domain)
- No visual flow builder — fully LLM-driven, no deterministic conversation paths
- Steep price jumps between tiers (Hobby $40 → Standard $150 = 275% increase)
- Free plan very limited (50 credits, basic models, 14-day inactivity deletion)
- Some user reports of billing issues and support responsiveness concerns
