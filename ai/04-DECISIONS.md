# 04 — Architecture Decision Log

## Purpose

This document records permanent architecture decisions for the OLSP.PROFITANDPRIVILEGE.COM repository.

It contains only active decisions. Historical or superseded decisions are removed.

---

# Active Decisions

- Architecture Freeze is active.
- Repository is the source of truth.
- Gold Master is the visual standard.
- OLSP Standard is the implementation standard.
- Builder owns presentation.
- Content Production owns editorial content only.
- Editorial QA validates but does not redesign.
- Publishing never modifies article content.
- Reuse existing implementation before creating anything new.
- Search first. Understand second. Modify last.
- One implementation task at a time.
- Validate every change before continuing.
